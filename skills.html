Minhas Habilidades

Como desenvolvedor frontend, possuo uma gama diversificada de habilidades que me permitem criar interfaces de usuário modernas, responsivas e funcionais.

Habilidades Técnicas

  • HTML5: Experiência na construção de estruturas semânticas e acessíveis, garantindo compatibilidade e SEO.
  • CSS3: Conhecimento avançado em layout com Flexbox e Grid, animações, transições e design responsivo.
  • JavaScript: Habilidade em criar interações dinâmicas, manipulação de DOM e desenvolvimento de aplicações SPA com frameworks modernos.
  • React: Desenvolvimento de componentes reutilizáveis, gerenciamento de estado com hooks e integração com APIs.
  • Versionamento: Uso do Git para controle de versão, colaboração e gerenciamento de código em projetos.
  • Design Responsivo: Adaptação de layouts para diferentes dispositivos e tamanhos de tela utilizando Media Queries.
  • UI/UX Design: Práticas de design centrado no usuário, prototipagem e uso de ferramentas como Figma para design de interfaces.

Ferramentas e Tecnologias

  • Git & GitHub: Controle de versão e colaboração em projetos.
  • Webpack: Empacotamento de módulos e otimização de recursos.
  • SASS: Pré-processador CSS para escrita de estilos mais eficientes e organizados.
  • Bootstrap: Framework para desenvolvimento rápido e responsivo de layouts.
  • VS Code: Editor de código com extensões personalizadas para aumento da produtividade.
